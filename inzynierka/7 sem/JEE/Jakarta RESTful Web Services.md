Model dojrzałości usług  (Richard Maturity Model)

- Level 0: Swamp of POX:
    - używa protokołu HTTP jedynie jako tunelu
    - SOAP
- Level 1: Resources:
    - definiuje zasoby,
        - path param - elementy ścieżki
        - query param - po znaku zapytania, typowe dla html
- Level 2: HTTP verbs:
    - wykorzystanie poprawnych metod HTTP,
    -  PUT - wstawienie zasobu (aktualizacja lub stworzenie),
    - GET - pobranie zasobu,
    - POST - dodanie zasobu do kolekcji,
    - DELETE - usunięcie zasobu.
- Level 3: Hypermedia controls:
    - wykorzystanie HATEOAS - Hypermedia As The Engine Of Application State
